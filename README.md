
# Unit Converter

Convert all kinds of units from Length and Time.

## Compiling for Linux

Requires [dotnet-sdk](https://dotnet.microsoft.com/en-us/download/dotnet/10.0).

```sh
make # compiling a build in ./bin/Debug, requires dotnet-sdk
make install # compiles a standalone executable and installs it systemwide
make uninstall # deletes the systemwide executable
make clean # cleans ./bin and deletes the systemwide executable
```
